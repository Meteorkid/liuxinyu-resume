### 2.2 鼠标跟随光晕 ✨

**灵感来源**：Linear、Vercel 等现代产品的交互设计

```javascript
// 鼠标跟随光晕
const glow = document.getElementById('cursorGlow');
let mx = 0, my = 0, gx = 0, gy = 0;

document.addEventListener('mousemove', (e) => {
  mx = e.clientX;
  my = e.clientY;
});

function animate() {
  gx += (mx - gx) * 0.08;  // 缓动跟随
  gy += (my - gy) * 0.08;
  glow.style.left = gx + 'px';
  glow.style.top = gy + 'px';
  requestAnimationFrame(animate);
}
```

**效果**：鼠标移动时，一个柔和的黄色光晕跟随移动，增添科技感。

### 2.3 3D 卡片倾斜效果 🎮

**灵感来源**：Apple 产品展示页的交互设计

```javascript
// 项目卡片 3D 倾斜
cards.forEach(card => {
  card.addEventListener('mousemove', (e) => {
    const rect = card.getBoundingClientRect();
    const x = e.clientX - rect.left;
    const y = e.clientY - rect.top;
    const centerX = rect.width / 2;
    const centerY = rect.height / 2;
    const rotateX = ((y - centerY) / centerY) * -8;
    const rotateY = ((x - centerX) / centerX) * 8;

    card.style.transform = `perspective(800px) rotateX(${rotateX}deg) rotateY(${rotateY}deg) scale(1.02)`;

    // 背景大字视差
    const bgText = card.querySelector('.project-card-bg-text');
    if (bgText) {
      const moveX = ((x - centerX) / centerX) * 15;
      const moveY = ((y - centerY) / centerY) * 10;
      bgText.style.transform = `translate(${moveX}px, ${moveY}px)`;
    }
  });
});
```

**效果**：鼠标在卡片上移动时，卡片产生 3D 倾斜，背景文字产生视差效果。

### 2.4 时间线滚动进度条 📊

**灵感来源**：Linear 的时间线设计

```javascript
// 时间线进度条随滚动变化
window.addEventListener('scroll', () => {
  const rect = timeline.getBoundingClientRect();
  const timelineTop = rect.top;
  const timelineHeight = rect.height;
  const windowHeight = window.innerHeight;

  const scrolled = windowHeight - timelineTop;
  const total = windowHeight + timelineHeight;
  const ratio = Math.max(0, Math.min(1, scrolled / total));

  progress.style.height = (ratio * timelineHeight) + 'px';
});
```

**效果**：随着页面滚动，时间线左侧的进度条逐渐填充，引导用户阅读。

### 2.5 数字计数器动画 🔢

**灵感来源**：数据可视化产品的统计展示

```javascript
// 获奖数量计数器
function update(now) {
  const elapsed = now - start;
  const progress = Math.min(elapsed / duration, 1);
  const eased = 1 - Math.pow(1 - progress, 3);  // easeOutCubic
  counter.textContent = Math.round(target * eased);
  if (progress < 1) requestAnimationFrame(update);
}
```

**效果**：当统计区域进入视口时，数字从 0 动态增长到目标值，使用 easeOutCubic 缓动函数。

### 2.6 磁吸卡片效果 🧲

**灵感来源**：Dribbble 上的微交互设计

```javascript
// 磁吸效果
const strength = 0.15;
cards.forEach(card => {
  card.addEventListener('mousemove', (e) => {
    const dx = e.clientX - cx;
    const dy = e.clientY - cy;
    const dist = Math.sqrt(dx * dx + dy * dy);
    const threshold = Math.max(rect.width, rect.height) * 0.6;

    if (dist < threshold) {
      const factor = 1 - dist / threshold;
      const moveX = dx * strength * factor;
      const moveY = dy * strength * factor;
      card.style.transform = `translate(${moveX}px, ${moveY}px)`;
    }
  });
});
```

**效果**：鼠标靠近卡片边缘时，卡片轻微向鼠标方向移动，产生磁吸感。

### 2.8 滚动进度条 📏

## 三、技术架构

### 3.1 前端技术栈
- **HTML5**：语义化标签，无障碍访问
- **CSS3**：CSS 变量、Flexbox/Grid 布局、动画
- **JavaScript**：原生 ES6+，无框架依赖

### 3.2 性能优化
- **懒加载**：IntersectionObserver 实现按需动画
- **被动事件监听**：`{ passive: true }` 提升滚动性能
- **requestAnimationFrame**：流畅的动画实现
- **CSS containment**：隔离重排范围

### 3.3 无障碍支持
- **prefers-reduced-motion**：自动跳过动画
- **语义化 HTML**：正确的标签结构
- **键盘导航**：支持 Tab 键访问
- **对比度**：符合 WCAG AA 标准
